# 交互问答（Intake）

用户丢来**链接或原文**、又没有一次给齐全部参数时，走本流程。  
目标：先对齐文章理解，再锁定变量，**得到用户回答之前不要生图、不要长篇 shot list**。

## 何时走问答

走问答：

- 用户给了链接 / 粘贴正文 / 文件，并说配图、插图、做图、试这个 skill 等
- 只给了文章，没写齐 IP / 风格 / 图上文字语言 / 张数 / 只要规划还是要出图

可跳过问答（用户明确说其一即可）：

- 「直接生成」「别问了」「用默认全包办」
- 一句话里已写齐关键变量，例如：`IP=note style=white-handdrawn label_lang=zh 生成5张 直接出图`

跳过时：缺的项用默认值，直接进入工作流。

## 步骤

### A. 读文

打开链接或读正文。读不到外链时，说明原因并请用户粘贴正文，**不要编造文章**。

### B. 确认理解（先发这一段，简短）

用固定小结构回复，便于用户点头或纠正：

```markdown
## 文章确认

- **标题/来源**：（若有）
- **一句话**：这篇文章在说什么
- **核心判断**：（1–3 条）
- **建议配图锚点**：（3–6 个短语，不展开成完整 shot list）
- **我读到的是否有误？** 有误请直接改；没问题进入下面选项。
```

### C. 问变量（同一条消息里问完，少轮次）

在文章确认后，紧跟选项表。标注**默认**，用户可回「默认」或只改某几项。

```markdown
## 请选变量（直接回复编号/id 即可）

1. **IP**（角色外形）
   - `xiaohei` 小黑 ← 默认
   - `note` 便签
   - `dot` 句号

2. **风格**
   - `white-handdrawn` 纯白手绘 ← 默认（目前仅此一项成熟）

3. **图上文字语言**（标注/手写批注用什么语言）
   - `zh` **中文（默认）** — 不选或回「默认」时一律用中文
   - `en` 英文
   - `bilingual` 中英双语（仍要短，每处尽量单词或 2–8 字，不要长句）
   - 说明：指**画面上的标签文字**，不是文章原文语言；角色名可按 IP 显示名或用户指定。
   - 未指定时**禁止**擅自改成英文。

4. **产出模式**
   - `shot-list` 只要配图规划，不生图
   - `generate` 规划并生成图片 ← 默认（若用户明确说先规划则用 shot-list）
   - `generate-only` 按约定直接生图，少写策略正文

5. **张数**
   - 短文建议 1–3；中长文 4–6；默认 **5**
   - 也可回具体数字 1–9（>9 时劝退到 ≤9）

6. **其他约束**（可选）
   - 例如：更怪一点 / 字更少 / 某段必须有图 / 某段不要图
   - 无则回「无」
```

用户也可一行答：

```text
IP=note，风格默认，label_lang=zh，5张，generate，无其他
```

或：`图上文字用英文` / `标注中英双语`

### D. 锁定后再执行

收到回答后：

1. 复述一行确认：`已锁定：IP=… · style=… · label_lang=… · 模式=… · 张数=…`
2. 加载对应 `personality` + `ips/<id>` + `styles/<id>`
3. shot list 与生图提示词中的标注必须使用锁定的 `label_lang`
4. 按模式执行：shot list 和/或 `image_gen`
5. **不要**在用户改口前擅自换 IP / 张数 / 图上语言

若用户只回「默认」或「ok / 没问题 / 继续」：全部用默认变量，模式 `generate`，张数 5，`label_lang=zh`。

### E. 改图轮次

用户已有图、只说编辑时：不必重新走完整 intake；只确认改什么即可。

## 默认速查

| 变量 | 默认 |
|------|------|
| IP | `xiaohei` |
| style | `white-handdrawn` |
| label_lang（图上文字） | `zh` |
| 模式 | `generate`（用户说先规划则 `shot-list`） |
| 张数 | `5` |
| 其他约束 | 无 |

## label_lang 执行要点

| 值 | 生图时 |
|----|--------|
| `zh` | 手写标注用中文；短词 2–8 字 |
| `en` | 手写标注用英文；短词 1–4 words，避免长句 |
| `bilingual` | 同一概念可用「中文 / English」极短并列，总标注处数仍 ≤5–8，不要变成对照表 |

无论哪种语言：少字、可读、禁止左上角类型大标题。
