# 风格菜单

用户指定风格时，只加载对应 `references/styles/<id>.md`。  
未指定时，默认 `white-handdrawn`。

| id | 显示名 | 一句话 | 何时选 |
|----|--------|--------|--------|
| `white-handdrawn` | 纯白手绘 | 纯白底、细黑线、红橙蓝短批注 | **默认**；最稳、最像历史示例 |
| `soft-paper` | 柔纸手绘 | 浅暖纸色、软铅笔感 | 想稍有温度、仍留白 |
| `notebook-pen` | 笔记本圆珠笔 | 蓝黑笔+红笔批注 | 方法/会议/学习笔记气质 |
| `risograph-duo` | 双色孔版 | 两色套印、轻颗粒 | 要一点印刷识别度 |
| `bold-marker` | 粗马克笔 | 更粗黑线、强剪影 | 远看也要认、结构要狠 |

## 解析规则

1. 用户写 `style=notebook-pen` / 「柔纸」「圆珠笔」「risograph」「马克笔」→ 映射上表。
2. 未指定 → `white-handdrawn`。
3. 未收录风格：用最接近的已有项，并说明未支持自定义整套新风格。
4. 不要一次加载全部风格文件。
5. 风格文件**只**管 Visual DNA；角色外形只在 IP 文件。

## 插图画风怎么选（简表）

| 文章气质 | 更合适 |
|----------|--------|
| 系统/工作流/默认 | `white-handdrawn` |
| 知识库/个人方法、略软 | `soft-paper` 或 `notebook-pen` |
| 品牌感/分享卡片 | `risograph-duo` |
| 结构对比要一眼看懂 | `bold-marker` |
