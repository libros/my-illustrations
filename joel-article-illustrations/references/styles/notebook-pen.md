# 风格: 笔记本圆珠笔（notebook-pen）

适合：方法论、会议、学习笔记气质的正文插图。

## 一句话

像在空白笔记本页上用蓝黑圆珠笔画的解释图，红笔偶尔批注，清爽、手写、不幼稚。

## 必须

- 16:9 横版。
- 背景：干净近白；可有**极淡**横线暗示（可有可无，不要抢主体）。
- 主线：蓝黑/深蓝圆珠笔手绘，线条微抖。
- 批注：红色圆珠笔为主强调；次要可用深蓝。
- 留白充足；一张一事。

## 颜色

- 主结构与角色：蓝黑笔。
- 重点/问题/结果：红笔。
- 路径/流程：可用稍亮的蓝或橙红，但整体像双色笔，不要彩虹。

## 英文 Visual DNA 块

```text
Clean near-white notebook page background, optional extremely faint ruling lines only if they stay quiet. Blue-black ballpoint pen hand-drawn line art, slightly wobbly. Sparse red ballpoint handwritten annotations for emphasis; secondary notes in blue. Lots of empty space. Looks like a thoughtful notebook doodle explaining a system, not a school worksheet, not a planner sticker page, no markers rainbow, no PPT, no realistic UI, no cute stickers.
```

## 绝对不要

- 手账贴纸、彩色荧光、和纸胶带装饰。
- 格子作业本堆满作业题感。
- 商业插画上色。
