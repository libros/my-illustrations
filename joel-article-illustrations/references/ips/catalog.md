# IP 菜单

用户指定 IP 时，只加载对应 `references/ips/<id>.md`。  
未指定时，默认 `xiaohei`。

| id | 显示名 | 一句话 | 适合 |
|----|--------|--------|------|
| `xiaohei` | 小黑 | 黑色实心荒诞工位员（默认） | 通用；与历史示例一致 |
| `note` | 便签 | 短命但关键的批注员 | 内容/编辑/标注感强的文章 |
| `dot` | 句号 | 负责收尾与闭环的冷抽象角色 | 判断、交付、结束、闭环 |
| `stick` | 火柴人 | 白板火柴棍小人 | 会议、流程、讲解感 |
| `cat` | 小猫 | 极简 deadpan 工位猫（不卖萌） | 轻内容、日常系统、仍要冷 |
| `dino` | 恐龙 | 袖珍简笔画工作龙（不 Q 萌） | 需要一点记忆点的隐喻 |

## 解析规则

1. 用户写 `IP=note` / `ip: dino` / 「用火柴人」「小猫」「恐龙」→ 选对应 id。
2. 用户写「小黑」或未提 IP → `xiaohei`。
3. 气质映射（可选）：
   - 批注/编辑感 → `note`
   - logo/收尾感 → `dot`
   - 白板讲解感 → `stick`
   - 轻日常但仍冷 → `cat`
   - 记忆点/反差体型 → `dino`
   - 不确定 → `xiaohei`
4. 不要一次加载全部 IP 文件。
5. 选定后，全文用该 IP 的**显示名**，不要混用其他 IP 名字。
6. `cat` / `dino` 必须读专属禁止：坚持 deadpan，禁止萌宠/Q 版。
