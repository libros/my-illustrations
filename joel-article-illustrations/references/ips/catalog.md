# IP 菜单

用户指定 IP 时，只加载对应 `references/ips/<id>.md`。  
未指定时，默认 `xiaohei`。

| id | 显示名 | 一句话 | 适合 |
|----|--------|--------|------|
| `xiaohei` | 小黑 | 黑色实心荒诞工位员（默认） | 通用；与历史示例一致 |
| `note` | 便签 | 短命但关键的批注员 | 内容/编辑/标注感强的文章 |
| `dot` | 句号 | 负责收尾与闭环的冷抽象角色 | 判断、交付、结束、闭环类文章 |

## 解析规则

1. 用户写 `IP=note` / `ip: dot` / 「用便签」「换句号」→ 选对应 id。
2. 用户写「小黑」或未提 IP → `xiaohei`。
3. 用户只描述气质时，可映射：
   - 要和批注一体、编辑感 → `note`
   - 要 logo 感、收尾感、极简抽象 → `dot`
   - 沿用原来 / 不确定 → `xiaohei`
4. 不要一次加载全部 IP 文件。
5. 选定后，全文用该 IP 的**显示名**，不要混用其他 IP 名字。
